# Dex Analytics

Dex Analytics is a real-time DEX analytics dashboard for OPN/IOPn Chain featuring token tracking, liquidity monitoring, swap analytics, price charts, and blockchain insights.

## Features

- Real-time token analytics
- Liquidity tracking
- Swap monitoring
- Price charts
- Trending pairs
- Wallet analytics
- DEX activity tracking

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Node.js
- ethers.js
- PostgreSQL

## Goal

Build a powerful analytics platform for the OPN/IOPn ecosystem similar to DexScreener and GeckoTerminal.
